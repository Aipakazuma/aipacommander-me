Position

Positionとは？

cssのプロパティです。ボックスの配置方法（基準位置）が、相対位置か絶対位置かを指定する際に使用します。

positon プロパティでは、配置方法を指定し、位置は top left right bottom と、併用して利用します。

使い方

static
特に配置方法を指定しません。この値のときには、 top bottom left right は適用されません。これが初期値です。
relative
相対位置への配置となります。 position: static を指定した場合に表示される位置が基準位置となります。
absolute
絶対位置への配置となります。親ボックスに position: static 以外の値が指定されている場合には、親ボックスの左上が基準位置となります。親ボックスに position: static 以外の値が指定されていない場合には、ウィンドウ全体の左上が基準位置となります。
fixed
絶対位置への配置となるのは position: absolute と同じですが、スクロールしても位置が固定されたままとなります。
