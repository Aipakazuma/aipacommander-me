Position

Positionとは？

cssのプロパティ。ボックスの配置方法（基準位置）が、相対位置か絶対位置かを指定する際に使用します。（なんのこっちゃ）

positon プロパティでは、配置方法を指定し、位置は top left right bottom と、併用して利用するとのこと

過去使ってきた経験

div で、囲んだ中で左上に要素を持っていきたいとか、windowの中で固定したいとかで使ったことがある。 人によっては、構造がメンテしづらくなるので、あまり使っちゃダメーって意見もあった（正しいかどうかはわからんけど）

使い方

static
初期値。 特に配置方法を指定しません。この値のときには、 top bottom left right は適用されない。 どんなの時に使うの？ レスポンシブとかで戻すときじゃない？
relative
相対位置への配置 position: static を指定した場合に表示される位置が基準位置となる。 position:relative した子要素で、他の position を使うイメージ
absolute
絶対位置への配置。親要素に position: static 以外の値が指定されている場合には、親要素左上が基準位置となります。親要素に position: static 以外の値が指定されていない場合には、ウィンドウ全体の左上が基準位置となる。
fixed
絶対位置への配置となるのは position: absolute と同じだけど、スクロールしても位置が固定されたままとなる。 何故かiosでバグがある。。。！（今回それを勉強するためにこのサイトを用意したといっても過言ではない）
